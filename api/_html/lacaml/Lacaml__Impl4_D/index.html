  • Parameter patt: default = `Full
  • Parameter uplo: default = whole matrix
  • Parameter b: The target matrix. By default a fresh matrix to accommodate the sizes m and n and the offsets br and bc is created.

  • Parameter incx: default = 1
  • Parameter scale: default = 0.
  • Parameter sumsq: default = 1.
